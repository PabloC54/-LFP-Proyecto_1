GUÍA DE USO

-Ejecutar el archivo simplesql.py, para esto es necesario Python.
En caso no se tuviera Python, se puede descargar en https://www.python.org/downloads/ .

-Se abrirá una linea '=>' de comandos, donde se pueden ejecutar los comandos propios de SimpleSQL. Para consultar los comandos se puede ejecutar 'ayuda'

La sintaxis de cada comando está especificada en 'ayuda'.
